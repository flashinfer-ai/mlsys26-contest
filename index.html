Contest Overview

🎯

The Challenge

Create AI agents that automatically generate optimized CUDA kernels for cutting-edge LLM operations. Your agent will receive kernel specifications and must produce high-performance code for NVIDIA Blackwell B200 GPUs.

📊

Benchmark

Compete across workloads derived from production models. Kernels are evaluated on correctness, speed, and win rate against FlashInfer baselines.

🔧

Platform

Submit and evaluate your kernels on FlashInfer-Bench (bench.flashinfer.ai). Registered teams receive compute credits for B200 access.

🤖

Two Approaches

We welcome both expert-crafted seed kernels with agent-assisted evolution, and fully agent-generated solutions. The two approaches will be evaluated separately. Agent solutions must open-source scripts to reproduce kernels. No API credits provided.

Competition Tracks

Three kernel categories targeting the most important operations in modern LLMs

Track A

MoE GEMM

Grouped GEMM kernels for Mixture-of-Experts models with low-precision and mixed-precision support.

Kernels moe_gemm_grouped, moe_gemm_fused, moe_scatter_gather
Precision FP16, BF16, FP8 (E4M3/E5M2)
Track B

Sparse Attention

Block-sparse attention kernels for efficient long-context processing with variable sparsity patterns.

Kernels deepseek_sparse_prefill, deepseek_sparse_decode
Features Variable block sizes, custom masks, paged KV
Track C

Gated Delta Net

Linear attention kernels with gating mechanisms for efficient sequence modeling.

Kernels gated_delta_net_fwd, gated_delta_net_recurrent
Features Linear complexity, state-space formulation
